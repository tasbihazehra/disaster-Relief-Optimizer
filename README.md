#Disaster Relief Route Optimizer (C++)

## Project Motivation
In the wake of the 2023 earthquakes in Turkey, it became clear that the biggest challenge for rescue teams is navigating damaged infrastructure. I developed this tool to simulate how **Dijkstraâ€™s Algorithm** can be used to find the safest and fastest routes for aid delivery when certain roads are blocked or destroyed.

##  Technical Implementation
* **Language:** C++
* **Algorithm:** Dijkstra's Shortest Path Algorithm.
* **Data Structure:** Adjacency Lists and Priority Queues (STL).
* **Logic:** The system treats cities as 'Nodes' and roads as 'Edges'. Each road is given a weight based on its damage level (1 = Clear, 10 = Blocked).
